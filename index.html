Список команд які ми розглянули на занняті з терміналом

  • mkdir - створити директорію
  • rm - відалити директорію
  • git init - стежити за змінами в папці
  • git status - переглянути зміни
  • git revert - операція відміни
  • git add - добавить до стеження
  • git commit - сберегти дію
  • git push - відправити зміни
  • git clone - скачати зміни з гітхабу на пк
